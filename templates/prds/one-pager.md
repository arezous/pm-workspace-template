**About This Template:**

This one-pager format is designed to:
- Force clarity and conciseness (one page when printed)
- Start with the problem, not the solution
- Focus on outcomes, not just outputs
- Embrace an experiment mindset

**When to use:**
- Early-stage feature exploration
- Pitching new ideas to stakeholders
- Aligning teams before detailed specs
- Fast-moving projects where speed matters
